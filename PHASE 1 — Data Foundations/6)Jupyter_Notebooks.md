# Step 6 — Jupyter / Notebooks

A notebook contains:

👉 Code  
👉 Output  
👉 Charts  
👉 Explanations  

All in one interactive document.

This is how professionals explore data.

---

## 1️⃣ What is Jupyter?

Notebook = cells.

Two types:

- Code cells → Run Python  
- Markdown cells → Write explanations  

It is like a lab notebook for data experiments.

---

## 2️⃣ Installation

```bash
pip install notebook
```

Run:

```bash
jupyter notebook
```

Browser opens → Create a new Python notebook.

---

## 3️⃣ Running Cells

Use:

```
Shift + Enter
```

Example:

```python
import pandas as pd

df = pd.read_csv("data.csv")
df.head()
```

Output appears immediately.

This allows interactive exploration.

---

## 4️⃣ Markdown Cells (Very Important)

Convert a cell to Markdown:

Press:

```
M
```

Example Markdown:

```markdown
## Salary Analysis
This dataset explores salary distribution across departments.
```

Now your notebook becomes a professional report.

Good analysts explain what they’re doing.

---

## 5️⃣ Proper Notebook Workflow

Always follow this structure:

1. Title + Objective  
2. Import libraries  
3. Load data  
4. Data Cleaning  
5. Exploratory Analysis  
6. Visualization  
7. Insights  
8. Conclusion  

This structure matters for interviews.

---

## 6️⃣ Why Notebooks Matter for Career

They are used for:

- Kaggle projects  
- Portfolio building  
- Technical interviews  
- Research experiments  
- Data storytelling  

Recruiters prefer seeing thinking process.
